免費手機獨立版一鍵上線

請雙擊「免費手機版一鍵上線.bat」。

助手會自動完成：
1. 安裝 Git 與 GitHub 官方工具。
2. 開啟 GitHub 官方登入頁。
3. 建立公開的 mobile-539-system 程式庫。
4. 上傳完整主程式、歷史資料庫與戰報。
5. 啟用免費 GitHub Pages。
6. 執行第一次雲端更新與運算。
7. 建立「手機獨立版網址.txt」並開啟手機網站。

你只需要完成一次 GitHub 官方登入確認。

注意：
- 免費版的程式、歷史資料庫、戰報與人氣來源皆為公開內容。
- 請勿在程式庫、人氣號碼表單或戰報放入姓名、電話、密碼等私人資料。
- 第一次上線通常需要等待 5 至 10 分鐘。
- 手機網站上可按「立即更新與重新運算」，以及提交 Facebook / 社團人氣號碼。
