# 539 手機完全獨立免費版

此公開程式庫使用 GitHub Actions 每日自動更新及運算，並使用 GitHub Pages 發布手機網站。

## 一鍵上線

在 Windows 雙擊 `免費手機版一鍵上線.bat`，依照 GitHub 官方登入頁完成一次登入。

## 手機使用

- 開啟 `https://你的GitHub帳號.github.io/mobile-539-system/`
- Android Chrome：選單 → 加到主畫面
- iPhone Safari：分享 → 加入主畫面
- 網站中的「立即更新」會開啟 GitHub Actions，可手動執行。
- 網站中的「提交人氣號碼」會開啟專用表單。

## 自動排程

每週一至週六台北時間約 `21:10`、`21:40`、`22:10` 嘗試更新。資料未更新完成時，不會使用舊資料發布新預測。
