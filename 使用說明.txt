539 主系統使用說明

1. 雙擊「點我一鍵啟動539主系統.bat」。
2. 系統會依序更新官方開獎資料、重新運算、結算上一期預測、執行完整性稽核，最後開啟中文網頁戰報。
3. 戰報中的日期分為：
   - 預測依據開獎日：運算時已知的最後一期開獎日期。
   - 預計開獎日：該筆預測所對應的下一次開獎日期。
   - 實際開獎日：結算時真正對應的開獎日期。
   - 建立時間與結算時間：系統實際執行與完成核對的時間。
4. 目前核心模式為「已恢復 2026-06-04 中4版 v31」。
5. 外部網站、Facebook、社群與直播推薦資訊已從正式主流程與戰報移除。
6. 手機同步：雙擊「點我手機版一鍵上線.bat」，系統會使用主程式內現有的發布腳本更新手機網站。
7. 自動排程：雙擊「點我安裝每日開獎後與開機自動更新.bat」，即使資料夾名稱含空格或括號也能安裝。
8. LINE 自動轉發：雙擊「點我設定LINE推送.bat」。建議選 broadcast，並把你自己的 LINE 加入該 LINE 官方帳號好友；或在 GitHub Secrets 設定 LINE_CHANNEL_ACCESS_TOKEN、LINE_DELIVERY_MODE、LINE_TO_ID。
   注意：「@開頭的官方帳號 ID」不是 Channel access token。Channel access token 是 LINE Developers Console 的 Messaging API 頁面產生的一長串授權字串。

注意：歷史命中不代表未來必然命中。本系統只提供統計研究與逐期真實結算。
